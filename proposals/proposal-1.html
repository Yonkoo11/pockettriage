An offline WHO IMCI triage assistant for community health workers.

Gemma 4 runs entirely on the device. The chart booklet on the wall, now in the worker's pocket — without the network.

Mustapha Alex · Medical intern, Nigeria Submitted to the Gemma 4 Good Hackathon 17 May 2026

Community health workers in Anambra, Maharashtra, and similar low-connectivity geographies use the WHO Integrated Management of Childhood Illness protocol every day. The protocol works; cross-referencing a paper booklet under time pressure does not. PocketTriage is the protocol as a fast, deterministic on-device tool — paediatric symptom in, Pink / Yellow / Green tier and a structured referral pathway out, with a non-bypassable safety layer that catches general danger signs even when the model is wrong. Nothing leaves the device.

Architecture

A single, narrow path — no server, no analytics, no fallback to anyone else's cloud.

CHW symptom text (+ optional photo) │ ▼ infer.py builds WHO-IMCI-grounded chat │ ▼ Ollama on 127.0.0.1:11434 → gemma4:e2b (5.1 B, Q4_K_M) │ ▼ raw model output (string) _extract_first_json + _validate_shape → tolerant JSON coerce │ ▼ { tier, pathway, reasoning, confidence } safety.apply_safety_layer R13 · danger-sign keyword → force Pink R14 · confidence < 0.4 → append "Escalate to medical officer" R15 · adult case → refusal │ ▼ Gradio renders tier card + pathway + reasoning + flags

Safety

Four invariants that the model cannot override.

PocketTriage treats the classifier as a powerful but fallible signal. The safety layer enforces these rules after the model speaks. 17 unit tests cover the layer (laptop/test_safety.py).

  • R13 · PinkIf the symptom or reasoning text contains any WHO IMCI general danger sign (unable to drink, vomits everything, convulsions, lethargic, unconscious, stiff neck, chest indrawing), the tier is forced to PINK even if the model said GREEN. Negation-aware: "no chest indrawing" does not fire.
  • R14 · FloorIf model confidence < 0.4, the pathway gains "Escalate to medical officer" regardless of tier. Low-confidence cases never bypass a human.
  • R15 · AdultAdult-specific terms (chest pain in adult, stroke in adult, pregnancy emergency) return a Refused result with referral to adult emergency protocol. Paediatric scope, hard boundary.
  • R16 · BannerA non-dismissable banner — "Decision support based on WHO IMCI 2014. Does NOT replace clinical judgment. Paediatric only (2 months – 5 years)" — renders as part of the layout, not as a closeable popup.

Run locally

Clean clone to first triage: about fifteen minutes.

# 1. Install Ollama (macOS shown)
brew install ollama
ollama serve &

# 2. Pull the model (one-time, ~7 GB)
ollama pull gemma4:e2b

# 3. Run PocketTriage
git clone https://github.com/yonkoo11/pockettriage
cd pockettriage/laptop
python3.12 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt

# 4. Verify Phase 1 Gate on the four canonical IMCI scenarios
python eval_runner.py

# 5. Launch the UI
python app.py   # → http://127.0.0.1:7860
